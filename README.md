# Alpha Camp Rails習題︰Photo Album

## 如何啟動？

```ruby
git clone git@github.com:victorloo/photo_album_practice2.git
bundle install
rake db:migrate
```

## 開發環境

* Ruby version: 2.4.3
* Rails version: 5.1.4

### 使用的 gem

* [carrierwave](https://rubygems.org/gems/carrierwave)
* [bootstrap-sass](https://rubygems.org/gems/bootstrap-sass)

## 使用者故事

### 照片
* 用戶可以瀏覽所有照片
* 用戶可以新增一筆照片
* 用戶可以瀏覽一筆照片
* 用戶可以修改一筆照片
* 用戶可以上傳照片
* 用戶可以刪除一筆照片
